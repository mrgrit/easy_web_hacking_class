# 🕵️ AI로 배우는 웹 해킹 특강 (Easy Web Hacking Class)

> **"코드 한 줄 몰라도, AI 에이전트와 함께라면 진짜 해커가 하는 일을 직접 해볼 수 있다."**

컴퓨터를 잘 모르는 **고등학생(비전공)** 을 위한 체험형 웹 해킹 특강입니다. 어려운 기술은
**AI 에이전트(Hermes Agent + 오픈 모델)** 가 대신 해주고, 학생은 **"이거 해보고 → 다음은 이거"**
절차만 따라가며 "우와!" 하는 결과를 직접 만들어 냅니다.

- 🎯 **대상**: 컴퓨터/코딩을 몰라도 됨. 호기심만 있으면 OK
- 🖱️ **방법**: **모든 실습을 브라우저(F12)만으로** 완주할 수 있게 설계 — `curl` 은 "기계가 하는
  같은 일"로 병기만 합니다
- 🤖 **도구**: **Hermes Agent**(오픈소스 터미널 에이전트) + **DGX Spark** 위의 **오픈 모델**
  → 학생 개인 계정·결제 불필요, 대화 내용이 학교 밖으로 나가지 않음
- 🧪 **결과**: 직접 만든 사이트, 직접 뚫는 웹, 친구들과 겨루는 미니 CTF, AI 챗봇 모의해킹

## 🌐 강좌 웹사이트 (포털)

이 강좌만을 위한 **웹사이트**가 있습니다. [`site/`](site/)

- 커리큘럼·교과서·실습을 브라우저에서 바로 읽기
- **회원가입 / 로그인** — 등급은 **학생 · 관리자** 두 가지
  - **정답·해설과 강사용 풀이는 관리자만** 볼 수 있습니다(학생 화면엔 HTML 자체가 없음)
  - 가입하면 **CTFd 계정도 같은 아이디·비밀번호로 자동 생성**됩니다(연동 설정 시)
- 📌 **담벼락** — 질문·발견·실패담을 메모지로 붙이는 Padlet 스타일 게시판
- 워크북·서약서 다운로드

```bash
cd infra && ./start.sh hub      # 강좌 사이트가 :8090 에 뜸
```

> **첫 가입자가 자동으로 관리자(admin)** 가 됩니다. 강사가 반드시 가장 먼저 가입하세요.
> 이후 가입자는 학생 등급이며, 가입 폼의 '관리자 코드'(`ADMIN_SIGNUP_CODE`)를 넣은 경우만
> 관리자가 됩니다.

## 🖥️ 실습 환경 — 중앙(hub) + 각자(victim)

**강의·CTF는 강사 서버 1대, 표적은 학생 PC 각자.** 표적을 한 대로 공유하면 한 학생이 비밀번호를
바꾸거나 스크립트를 심는 순간 반 전체 실습이 망가지기 때문입니다.
자세한 배치·이유·운영 팁은 **[`docs/DEPLOYMENT.md`](docs/DEPLOYMENT.md)**.

**🚀 맨바닥에서 한 방 (리눅스만 깔린 상태):**
```bash
./setup.sh hub          # 강사 서버: 강좌사이트 + CTFd(+문제 자동등록) + AI도우미
./setup.sh victim       # 학생 PC : DVWA(자동초기화) + NeoBank + MediForum + AICompanion
./setup.sh              # 한 대에서 전부 (혼자 학습할 때)
```

**이미 Docker 가 있으면:**
```bash
cd infra
./start.sh hub          # 또는  ./start.sh victim  /  ./start.sh  /  ./start.sh extras
```

| 사이트 | 주소 | 어디에 | 쓰는 곳 |
|--------|------|--------|---------|
| 강좌 사이트 | `:8090` | 강사 서버 | 전 주차 포털·회원·담벼락 |
| CTFd | `:8000` | 강사 서버 | Week 05 CTF 플랫폼 |
| AI 도우미 | `:8001` | 강사 서버 | Week 05 힌트 챗봇 |
| DVWA | `:8088` | 학생 PC | Week 03 웹 해킹 |
| NeoBank | `:3001` | 학생 PC | Week 04 AI 모의해킹 |
| MediForum | `:3003` | 학생 PC | Week 05 CTF 표적 |
| AICompanion | `:3005` | 학생 PC | 특별 세션 표적 |

**AI 두뇌**는 외부 **DGX Spark** 의 Ollama(`:11434`) 를 반 전체가 공유합니다.
학생 PC 의 Hermes Agent 가 그리로 질문을 보냅니다.

## 🏁 mini-CTF (Week 05)

- CTFd 기반: **회원가입 / 실시간 리더보드 / AI 도우미 연동**
- **6문제 전부 브라우저 + F12 만으로 풀립니다.** 주소를 "찍어서 맞히는" 문제는 없습니다 —
  `robots.txt`, Network 탭, 화면의 버튼이 다음 단계를 알려 줍니다.
- 문제·정답 일괄 등록: [`ctf/`](ctf/)
- ★ **수업 전 자가 점검 필수:**
  ```bash
  cd ctf && python3 verify_ctf.py --victim <표적IP> --ctfd http://<hub-ip>:8000 --token <TOKEN> --submit
  ```
  표적을 실제로 6문제 다 풀어 보고 → 정답표와 대조하고 → CTFd 등록 상태를 확인하고 →
  테스트 계정으로 제출까지 해 봅니다. 전부 ✓ 면 "찾았는데 오답" 사고가 나지 않습니다.
- **강사 전용 상세 풀이**: [`solutions/`](solutions/) — 강좌 사이트에서 **admin 로그인 시 `/solutions`**
